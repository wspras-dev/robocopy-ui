## üöÄ Cara Menggunakan

### Contoh 1: Copy dengan Preserve Junctions
1. Buka aplikasi
2. Set Source dan Destination
3. Buka tab **"Junction & Links"**
4. Centang **"/SJ - Salin Junction/Symbolic Link itu sendiri"**
5. Klik "Run Robocopy"

Generated Command:
```
robocopy "C:\Source" "D:\Backup" *.* /SJ /R:1 /W:30 ...
```

### Contoh 2: Exclude Junctions (Avoid Infinite Loops)
1. Buka tab **"Junction & Links"**
2. Centang **"/XJ - Kecualikan semua Junction points"**
3. Klik "Run Robocopy"

Generated Command:
```
robocopy "C:\Source" "D:\Backup" *.* /XJ /R:1 /W:30 ...
```

### Contoh 3: Copy Hanya File Junction
1. Buka tab **"Junction & Links"**
2. Centang **"/XJF - Kecualikan Junction untuk File"**
3. Klik "Run Robocopy"

Generated Command:
```
robocopy "C:\Source" "D:\Backup" *.* /XJF /R:1 /W:30 ...
```
